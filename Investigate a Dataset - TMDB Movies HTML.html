Introduction

Dataset Description

The TMDb movies dataset contains movie-level metadata including:

  • id - unique movie identifier
  • original_title - movie title
  • imdb_id - IMDB id
  • popularity - TMDb popularity score
  • budget - production budget
  • revenue - worldwide revenue
  • genres - pipe-separated genre labels (Action|Adventure) or JSON-like lists in some variants
  • release_date - release date in various formats
  • vote_count - number of TMDb votes
  • vote_average - average TMDb rating
  • runtime - movie runtime in minutes
  • release_year - expected release year (may contain 2-digit YEAR issues)

Other columns may be present. We will use budget, revenue, genres, release_date / release_year, vote_average, and popularity mostly

def format_plot(title, xlabel, ylabel):
    """Apply consistent labeling and tight layout for matplotlib plots."""
    plt.title(title, fontsize=12, fontweight='bold')
    plt.xlabel(xlabel, fontsize=10)
    plt.ylabel(ylabel, fontsize=10)
    plt.grid(True)
    plt.tight_layout()

def clean_numeric(df, col, treat_zero_as_na=False):
    """Convert a column to numeric (coerce errors) and optionally treat zeros as NaN."""
    df[col] = pd.to_numeric(df[col], errors='coerce')
    if treat_zero_as_na:
        df.loc[df[col] == 0, col] = np.nan
    return df

def parse_genre_field(val):
    """Return a list of genre names parsed from various genre formats."""
    if pd.isna(val):
        return []
    s = str(val).strip()
    # Pipe-separated
    if '|' in s and not s.startswith('['):
        return [p.strip() for p in s.split('|') if p.strip()]
    # JSON/list-of-dicts style
    if s.startswith('[') and ('name' in s or '{' in s):
        try:
            cleaned = s.replace('null', 'None')
            parsed = ast.literal_eval(cleaned)
            names = []
            for item in parsed:
                if isinstance(item, dict):
                    if 'name' in item:
                        names.append(item['name'])
                    elif 'title' in item:
                        names.append(item['title'])
                elif isinstance(item, str):
                    names.append(item)
            return [n for n in names if n is not None]
        except Exception:
            # fallback regex
            names = re.findall(r"'name'\s*:\s*'([^']+)'", s)
            return names
    # Comma-separated
    if ',' in s and not s.startswith('['):
        return [p.strip() for p in s.split(',') if p.strip()]
    # Single genre string
    if s:
        return [s]
    return []

def fix_year(y):
    """
    Fix two-digit years and keep four-digit years.
    Logic:
      - if y is NaN -> NaN
      - if 0 <= y <= 25 -> 2000 + y (years 2000-2025)
      - if 26 <= y <= 99 -> 1900 + y (years 1926-1999)
      - else (>=100) -> y (assume already 4-digit)
    """
    try:
        if pd.isna(y):
            return np.nan
        y = int(y)
        if 0 <= y <= 25:
            return 2000 + y
        if 26 <= y <= 99:
            return 1900 + y
        return y
    except Exception:
        return np.nan

Data Cleaning

I will be cleaning the data set by:

  1. Convert budget and revenue to numeric and treat zeros as missing values (these are often placeholders).
  2. Parse release_date into datetime and extract release_year. Fix 2-digit years ("97" 1997).
  3. Parse the genres column into a list and explode rows to get one (movie, genre) per row.
  4. Create profit = revenue - budget for profitability analysis (where values exist).
  5. Drop unnecessary duplicates if present.
  6. Fixing Release Year Issues

df = df.copy()

if 'release_date' in df.columns:
    df['release_date_parsed'] = pd.to_datetime(df['release_date'], errors='coerce')
    # first try to extract year from parsed date
    df['release_year_from_date'] = df['release_date_parsed'].dt.year
else:
    df['release_date_parsed'] = pd.NaT
    df['release_year_from_date'] = np.nan

# some datasets already have a release_year column  use it as fallback
if 'release_year' in df.columns:
    # coerce to numeric where possible
    df['release_year_raw'] = pd.to_numeric(df['release_year'], errors='coerce')
else:
    df['release_year_raw'] = np.nan

# Build final release_year: prefer parsed date year, otherwise use raw, then fix 2-digit issues
df['release_year_candidate'] = df['release_year_from_date'].fillna(df['release_year_raw'])
df['release_year'] = df['release_year_candidate'].apply(fix_year)

# Report range
print("Release year: min =", int(np.nanmin(df['release_year'])), "max =", int(np.nanmax(df['release_year'])))

# 3) Parse genres into list and explode

Fixing Release Year Issues

The original dataset included several incorrect release years, such as values like 72, 95, 976, and even 0. These errors caused problems during visualization, especially when plotting budget or revenue over time, since two-digit years were interpreted as future years (e.g., 72 2072).

To correct this, I applied the following cleaning steps:

  • Converted all two-digit years (199) into the 1900s (e.g., 72 1972).
  • Removed movies with unrealistic release years before 1900 or after 2025.
  • Ensured all remaining release_year values were valid for timeline visualizations.

This cleanup prevented incorrect spikes in the graphs and ensured that trends over time accurately reflected real movie release patterns.

Conclusions & Limitations

This analysis explored trends in movie popularity and ratings using the TMDB dataset. Several clear patterns emerged. Movie production has steadily increased over the years, with a noticeable rise beginning in the early 2000s. When examining genre performance, Documentary and War films tended to receive the highest average ratings, suggesting that audiences respond strongly to more serious or information driven storytelling. Other popular genres, like Adventure and Action, showed strong viewer engagement but slightly lower average ratings.

There are important limitations to consider. Many movies had missing or incomplete data, especially around release year or genre, which limited the number of films that could be included in some parts of the analysis. Ratings can also be influenced by factors not reflected in the dataset, such as marketing, cultural trends, or reviewer demographics. Additionally, because the dataset only includes movies listed on TMDB, it may not fully represent every film released globally.

Even with these limitations, the analysis provides useful insight into how movie ratings vary across genres and how film production trends have evolved over time. A more complete dataset would allow for deeper and more detailed exploration.
